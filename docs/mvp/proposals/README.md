## Lifecycle Rules

1. Each MVP doc has one source-of-truth status: `draft`, `in_progress`, `complete`, or `deferred`.
2. Proposal section defines scope, constraints, and success gates.
3. Disposition section records outcome, evidence links, and explicit next action.
4. Any gate closure must link to reproducible evidence artifacts under `docs/mvp/evidence/`.
